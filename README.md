# Sales Decline Analysis

## Project Overview
This project analyzes sales decline across regions, customer types, and sales channels. 
Python is used for data cleaning and exploratory analysis, and Power BI is used to build an interactive dashboard.

## Dataset
- sales_decline_dataset.csv – raw sales data
- sales_cleaned_for_powerbi.csv – cleaned dataset prepared using Python

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI Desktop

## Project Workflow
1. Data loading and inspection
2. Data cleaning and feature engineering
3. Exploratory Data Analysis (EDA)

## Key Insights
- Sales decline is driven mainly by specific regions
- Returning customers show stronger decline than new customers
- Online sales are more stable than offline sales

## Power BI Dashboard
The dashboard includes:
- KPI cards (Total Sales, Orders, Average Order Value)
- Monthly sales trend
- Sales by region
- Sales by customer type
- Sales by channel
- Interactive slicers
